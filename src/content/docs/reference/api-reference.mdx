## Component Functions

### `defineComponent(options)`

Creates a `ComponentDefinition` from a Zod schema and metadata. This is the primary API for registering custom components.

```typescript
import { defineComponent } from '@coralstack/uibuilder'

const definition = defineComponent({
  type: 'MyComponent',
  label: 'My Component',
  category: 'Content',
  schema: mySchema,
  defaultProps: { title: 'Hello' },
})
```

**Parameters:**

| Option | Type | Required | Default | Description |
|--------|------|----------|---------|-------------|
| `type` | `string` | Yes | — | Unique component type identifier |
| `label` | `string` | Yes | — | Display name in the sidebar |
| `category` | `string` | Yes | — | Sidebar category grouping |
| `description` | `string` | No | — | Tooltip description |
| `schema` | `ZodObject<T>` | Yes | — | Zod schema defining props |
| `defaultProps` | `Record<string, unknown>` | Yes | — | Default values for new instances |
| `allowsChildren` | `boolean` | No | `false` | Whether this is a container |
| `layoutType` | `ContainerLayoutType` | No | — | CSS layout mode for drop zones |
| `commonProps` | `CommonPropGroup[] \| false` | No | `['visual', 'sizing']` | Common prop groups |
| `editorConfig` | `EditorConfig` | No | — | Resize handles and constraints |

**Returns:** `ComponentDefinition`

### `buildCommonStyles(props)`

Converts common prop values (prefixed with `_`) into a CSS style object.

```typescript
import { buildCommonStyles } from '@coralstack/uibuilder'

const styles = buildCommonStyles({
  _borderRadius: '8px',
  _opacity: '0.9',
  _width: '300px',
})
// → { borderRadius: '8px', opacity: 0.9, width: '300px' }
```

**Parameters:** `Record<string, unknown>` — the component's full props object

**Returns:** `React.CSSProperties`

## Data Binding Functions

### `$path` — Path Expressions

#### `isPathExpression(value)`

Checks whether a value is a `$path` binding expression.

```typescript
isPathExpression({ $path: '/user/name' }) // true
isPathExpression('hello')                 // false
```

**Returns:** `boolean`

#### `createPathExpression(path)`

Creates a `$path` binding expression object.

```typescript
createPathExpression('/user/name')
// → { $path: '/user/name' }
```

**Returns:** `PathExpression`

#### `resolveBindingValue(value, stateModel)`

Resolves a single value against a state model. Handles all expression types (`$path`, `$cond`, `$eq`, `$each`, `$computed`, `$bind`, `$fetch`, `$transform`).

```typescript
const state = { user: { name: 'Jane' } }
resolveBindingValue({ $path: '/user/name' }, state) // 'Jane'
resolveBindingValue('static text', state)            // 'static text'
```

**Returns:** `unknown`

#### `resolveBindingsForDesignTime(props, stateModel)`

Resolves all binding expressions in a props object for design-time display.

```typescript
resolveBindingsForDesignTime(
  { text: { $path: '/user/name' }, level: 'h1' },
  { user: { name: 'Jane' } }
)
// → { text: 'Jane', level: 'h1' }
```

**Returns:** `Record<string, unknown>`

#### `enumerateStatePaths(obj, prefix?)`

Lists all leaf paths in a state model for use in path picker UIs.

```typescript
const state = { user: { name: 'Jane', age: 30 }, items: [1, 2] }
enumerateStatePaths(state)
// [
//   { path: '/user/name', type: 'string', preview: '"Jane"' },
//   { path: '/user/age', type: 'number', preview: '30' },
//   { path: '/items', type: 'array', preview: 'Array(2)' },
// ]
```

**Returns:** `StatePathEntry[]`

#### `resolveStringInterpolation(template, stateModel)`

Resolves `${$path:...}` template expressions inside strings.

```typescript
resolveStringInterpolation(
  'Hello, ${$path:/user/name}!',
  { user: { name: 'Jane' } }
)
// → 'Hello, Jane!'
```

**Returns:** `unknown`

### `$cond` — Conditional Expressions

#### `isCondExpression(value)`

Checks whether a value is a `$cond` conditional expression.

**Returns:** `boolean`

#### `createCondExpression(test, thenValue, elseValue)`

Creates a `$cond` expression.

```typescript
createCondExpression({ $path: '/isAdmin' }, 'admin', 'user')
// → { $cond: { test: { $path: '/isAdmin' }, then: 'admin', else: 'user' } }
```

**Returns:** `CondExpression`

### `$eq` — Equality Expressions

#### `isEqExpression(value)`

Checks whether a value is a `$eq` equality expression.

**Returns:** `boolean`

#### `createEqExpression(left, right)`

Creates a `$eq` expression.

```typescript
createEqExpression({ $path: '/status' }, 'active')
// → { $eq: [{ $path: '/status' }, 'active'] }
```

**Returns:** `EqExpression`

### `$each` — List Expressions

#### `isEachExpression(value)`

Checks whether a value is a `$each` list expression.

**Returns:** `boolean`

#### `createEachExpression(source, itemTemplate?, keyPath?)`

Creates a `$each` expression.

```typescript
createEachExpression('/items', { name: { $item: 'name' } }, 'id')
```

**Returns:** `EachExpression`

#### `resolveEachExpression(expr, stateModel)`

Resolves a `$each` expression against state. Returns an array of resolved items.

**Returns:** `unknown[]`

### `$computed` — Computed Expressions

#### `isComputedExpression(value)`

Checks whether a value is a `$computed` expression.

**Returns:** `boolean`

#### `resolveComputedValue(expression, stateModel)`

Evaluates a JS expression with state keys as variables.

```typescript
resolveComputedValue('revenue * 1.1', { revenue: 48500 })
// → 53350
```

**Returns:** `unknown`

### `$bind` — Two-Way Binding

#### `isBindExpression(value)`

Checks whether a value is a `$bind` expression.

**Returns:** `boolean`

#### `createBindExpression(path, event?, transform?)`

Creates a `$bind` expression.

```typescript
createBindExpression('/form/email', 'onChange', 'string')
```

**Returns:** `BindExpression`

#### `resolveBindExpression(expr, stateModel, updateState)`

Resolves a `$bind` expression. Returns `{ value, onChange }`.

**Returns:** `{ value: unknown; onChange: (newValue: unknown) => void }`

### `$handler` — Event Handlers

#### `isHandlerExpression(value)`

Checks whether a value is a `$handler` expression.

**Returns:** `boolean`

#### `createHandlerExpression(actions)`

Creates a `$handler` expression.

```typescript
createHandlerExpression([
  { type: 'setState', path: '/counter', value: 0 },
  { type: 'navigate', pageIndex: 2 },
])
```

**Returns:** `HandlerExpression`

#### `resolveHandlerExpression(actions, stateModel, updateState, navigatePage?)`

Resolves handler actions into a callback function.

**Returns:** `() => void`

### `$fetch` — API Fetch

#### `isFetchExpression(value)`

Checks whether a value is a `$fetch` expression.

**Returns:** `boolean`

#### `createFetchExpression(url, options?)`

Creates a `$fetch` expression.

```typescript
createFetchExpression('https://api.example.com/data', {
  method: 'GET',
  fallback: [],
  cacheDuration: 60000,
})
```

**Returns:** `FetchExpression`

#### `resolveFetchUrl(url, stateModel)`

Resolves `${$path:...}` interpolations in a fetch URL.

**Returns:** `string`

### `$transform` — Transform Pipes

#### `isTransformExpression(value)`

Checks whether a value is a `$transform` expression.

**Returns:** `boolean`

#### `createTransformExpression(source, pipes)`

Creates a `$transform` expression.

```typescript
createTransformExpression(
  { $path: '/user/name' },
  [{ op: 'uppercase' }, { op: 'truncate', args: [20] }]
)
```

**Returns:** `TransformExpression`

#### `applyTransforms(value, pipes)`

Applies a chain of transform operations to a value.

**Returns:** `unknown`
