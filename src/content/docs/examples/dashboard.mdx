This example shows how to build a Grafana-style dashboard with custom chart, stat card, and text components alongside UIBuilder's built-in primitives.

## Configuration

```typescript
// .uibuilder/config.ts
import type { UIBuilderConfig } from '@coralstack/uibuilder'

const config: UIBuilderConfig = {
  components: ['./components/**/*.catalogue.tsx'],
  categoryOrder: ['Layout', 'Typography', 'Actions', 'Forms', 'Data', 'Charts', 'UI'],
  alias: { '@/': './' },
  css: ['./app.css'],
  jsonRender: {
    catalog: { path: './lib/render/catalog.ts', exportName: 'dashboardCatalog' },
    registry: { path: './lib/render/registry.tsx', exportName: 'registry' },
  },
  initialState: {
    metrics: {
      totalRevenue: 48520,
      totalCustomers: 1243,
      activeSubscriptions: 876,
      churnRate: 2.4,
      growthPercent: 12.5,
      mrr: 38750,
    },
    revenueChart: {
      title: 'Monthly Revenue',
      data: 'Jan:32000,Feb:35000,Mar:33500,Apr:38000,May:41200,Jun:48520',
    },
    user: { name: 'Ahmed', role: 'Admin', email: 'ahmed@coralstack.com' },
    // ... more sample data for customers, invoices, alerts, etc.
  },
}

export default config
```

The `initialState` provides sample dashboard data that components can bind to via expressions like `{ "$path": "/metrics/totalRevenue" }`. This state is available immediately when the editor loads — no need to manually enter it in the Page State panel.

## Custom Components

### DashboardGrid (Root Layout)

A CSS grid container used as the root layout. Configurable columns and gap.

```tsx
export function DashboardGrid({ columns = 12, gap = 'md', children }) {
  return (
    <div style={{
      display: 'grid',
      gridTemplateColumns: `repeat(${columns}, 1fr)`,
      gap: GAP_MAP[gap],
      padding: GAP_MAP[gap],
      minHeight: '100vh',
      backgroundColor: '#f9fafb',
    }}>
      {children}
    </div>
  )
}
```

### Chart (CSS-Only)

Supports bar, donut (conic-gradient), and sparkline (SVG) chart types — no external charting dependency needed.

```typescript
const chartSchema = z.object({
  title: z.string().default('Chart'),
  chartType: z.enum(['bar', 'donut', 'sparkline']).default('bar'),
  values: z.string().default('[65, 40, 85, 30, 55]'),
  labels: z.string().default('["Mon", "Tue", "Wed", "Thu", "Fri"]'),
  color: z.string().default('#3b82f6'),
  height: z.number().default(200),
})
```

### StatCard

Big metric number with label and trend indicator.

### TextCell

Simple text display with variant (heading, body, caption) and alignment.

## Running

```bash
cd examples/dashboard
yarn uibuilder
```

The editor opens with:
- **DashboardGrid** as the root element
- All 4 dashboard components in the sidebar under Charts/Data categories
- All 16 built-in components also available
- Drag charts and stat cards onto the grid to build your dashboard

<Aside type="tip">
The dashboard components use CSS-only charts (no external dependencies). Bar charts use flexbox heights, donut charts use `conic-gradient`, and sparklines use inline SVG.
</Aside>
